# homebrew-nicotine-plus

Homebrew cask tap for [Nicotine+](https://nicotine-plus.org/) nightly macOS builds.

## Install

```sh
brew install radiosilence/nicotine-plus/nicotine-plus-nightly
```

Or add the tap first:

```sh
brew tap radiosilence/nicotine-plus
brew install nicotine-plus-nightly
```

## Update

```sh
brew upgrade nicotine-plus-nightly
```

## Uninstall

```sh
brew uninstall nicotine-plus-nightly
brew untap radiosilence/nicotine-plus
```

## Notes

- Requires macOS Ventura or later
- Supports both Apple Silicon and Intel Macs
- Builds are updated nightly from the [nicotine-plus](https://github.com/radiosilence/nicotine-plus) fork
